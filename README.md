# 🚀 tdx_quant: 通达信本地二进制极速量化数据与选股系统

`tdx_quant` 是一个基于 WSL (Ubuntu/Linux) 环境构建的**高性能单机量化交易与数据系统**。系统通过直接高速解析本地通达信（TDX）客户端的二进制日K线及全局 GBBQ 股本变迁权息数据库，利用 **Snappy Parquet** 进行高性能持久化，并结合 **DuckDB** 实现亚秒级的多线程“板块资金集聚共振突破”因子选股引擎。

---

## ✨ 核心优势与技术亮点

1. **极速内存映射解析**：
   - 利用 NumPy 内存结构体直接映射通达信 `.day` 二进制文件，**解析单股 20 年历史日K线仅需约 1 毫秒**。
   - 完美解密并重构了通达信 GBBQ 股本变动库，提取全历史权息数据进行**前复权价格及成交量（Volume）精准修正**。
2. **多进程增量同步**：
   - `sync_market.py` 自动匹配 CPU 物理核心数进行并行加速，首次全量同步 **9,185 个文件仅需 83 秒**。
   - 引入 mtime 修改时间比对算法，再次运行只需 **20~40 秒** 即可极速完成全市场增量对齐。
3. **二进制板块映射与智能检索**：
   - 完美破解并对齐了板块文件 `block_gn.dat` (概念)、`block_fg.dat` (风格)、`block_zs.dat` (指数) 的 **2813 字节固定块记录结构**。
   - 重构了 `query.py` 命令行检索工具，支持模糊检索板块、展现板块内全部个股、以及在查询个股K线时高颜值展示该股归属的所有板块。
4. **DuckDB 因子共振选股**：
   - 基于 DuckDB 分析型列式引擎，**在 7 秒多内并发扫描全市场千万级日线数据并完成复杂的 Window Functions 窗口运算**。
   - 独创“自下而上资金宽度共振”选股算法，锁定行业风口集聚爆发的多头主线，自动导出精美 Markdown 选股报告。

>
> ```python
> # -------------------------------------------------------------
> # 路径配置：若通达信安装目录发生变化，仅需修改此处的 TDX_DIR 变量即可 (WSL 挂载绝对路径)

> TDX_DIR = "/your/new/tdx/path"  # 填入你新的通达信安装路径 (例如 "/mnt/d/new_tdx")

## 🛠️ 环境准备与快速开始

### 1. 安装基础依赖
系统在运行选股器时会**静默自动检测并为你配置 DuckDB**。你也可以手动安装所需的基础数据科学库：
```bash
pip install numpy pandas pyarrow duckdb
```

### 2. 核心命令与使用场景

#### 🚀 步骤一：一键全市场极速同步 (K线与板块)
该脚本会自动加载股本数据库、扫描通达信目录、进行增量 mtime 比对，并完成所有板块的自动映射和更新：
```bash
python3 sync_market.py
```
> **提示**：首次运行会建立全市场 9,000+ 个 Parquet 缓存；往后每日收盘后运行，只需约 30 秒即可完成增量更新。

#### 🔍 步骤二：个股/板块高颜值检索
`query.py` 提供了极强的人性化终端查询功能：

* **查询单股日K线及所属板块**（默认显示最新10行，可传入参数限制行数）：
  ```bash
  python3 query.py 600000 5
  ```
* **模糊查询板块名称**（支持中文关键字）：
  ```bash
  python3 query.py 汽车
  ```
  *(会列出：华为汽车、汽车芯片、小米汽车、飞行汽车等，并提示完整查询指令)*
* **查询板块下的所有成分股**：
  ```bash
  python3 query.py 飞行汽车
  ```
* **一键扫盘大类板块**：
  ```bash
  python3 query.py concept   # 列出全市场 269 个概念板块
  ```

#### 🏆 步骤三：一键多线程 SQL 因子共振选股
直接运行选股器，系统会利用你电脑的所有 CPU 核心在几秒内压榨式扫描千万级数据，并输出当天最抢眼的“共振突破股”：
```bash
python3 screener.py
```
选股完成后，会**自动在项目根目录下刷新并保存 [breakout_report.md](file:///home/liliiflora/work/wsl-agy-projects/tdx_quant/breakout_report.md)** 报告！

## 📊 因子选股策略释义

系统采用的**“自下而上资金共振突破宽度（Sector Breadth Breakout）”**模型，公式包含以下三大维度：

1. **个股价格突破**：
   $$\text{Close\_Adj} > \text{MA}(20)$$
   *个股最新前复权收盘价站上 20 日生命均线，趋势由弱转强。*
2. **成交放量加速**：
   $$\text{Volume} > 1.5 \times \text{Vol\_MA}(5)$$
   *今日成交量放大到 5 日均量的 1.5 倍以上，显示有主力增量资金大笔抢筹。*
3. **行业风口共振**：
   $$\text{Block\_Breakout\_Count} \ge 3$$
   *统计个股所属的概念板块中，今日同时触发“价格突破 + 放量加速”的个股总数。**只保留所属板块今日至少有 3 只股票同时突破的个股，并实时算出占比（突破数 / 该板块总股数）**。*

这一策略能完美过滤掉冷门股，**只买在资金抱团主线上**，让你一眼看清当前全市场的最热风口以及绝对的龙一龙二。
